# MontyPyHall

An (unnecessarily) multi-processed simulator of the [Monty Hall Problem](https://en.wikipedia.org/wiki/Monty_Hall_problem)
written in Python.

### Install & Run

```
git clone https://github.com/dmod/MontyPyHall.git
cd MontyPyHall/
python mhsim
```
